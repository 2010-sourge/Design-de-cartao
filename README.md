# Product Card - iPhone 12

Projeto de prática de front-end que consiste em um **card de produto** estilizado, simulando uma vitrine de e-commerce para o iPhone 12.

## 📋 Sobre o projeto

Página simples desenvolvida com HTML e CSS que exibe um cartão de produto com nome, avaliação por estrelas, descrição, imagem e botão de compra, sobre um fundo gradiente em tons de rosa/magenta.

## 🖼️ Preview

O card de produto exibido na página possui:

- **Título do produto:** iPhone 12
- **Avaliação:** sistema de estrelas (4.5 de 5)
- **Descrição:** texto placeholder (Lorem Ipsum)
- **Imagem do produto:** iPhone 12 em três cores
- **Botão de ação:** "Buy Now" com preço de $1225

## 🛠️ Tecnologias utilizadas

- HTML5
- CSS3 (gradiente de fundo, sombras, bordas arredondadas)
- Ícones de estrela (Font Awesome ou unicode)

1. Clone ou baixe este repositório.

3. Utilize uma extensão como o **Live Server** (VS Code) para rodar em ambiente local, por exemplo:
   ```

   ```
